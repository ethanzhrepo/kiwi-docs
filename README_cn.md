## 项目介绍

Kiwi 支付平台是一个现代化的支付解决方案，支持：

- 一次性支付和周期性订阅服务
- 多种支付方式集成
- 完整的商户管理系统
- 强大的 CLI 工具支持
- 完善的 API 接口
